Welcome to A Game of Boards

What is Game of Boards?
 A Game of Boards is an online board game that is a mix between the tv show Game of Thrones and the board game Snakes and Ladders.

How to play
First you select a character -token- for your self. After you have selected your character, you select the character -token- for the computer.

Cookies
This game uses cookies, so please make sure that your web browser allows the use of cookies.

Good luck and have fun!
