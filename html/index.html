Welcome to A Game of Boards

What is Game of Boards?
 A Game of Boards is an online board game that is a mix between the tv show Game of Thrones and the board game Snakes and Ladders.

Tutorial
If you have never played this game before, we suggest that you do a quick run through of the tutorial by clicking the 'Run Tutorial' Button.

Skip tutorial
If you know how to play or don't want to run the tutorial, you can press the 'Skip Tutorial' button. You can always run the tutorial later by clicking the info button in the top right corner.

Cookies
This game uses cookies, so please make sure that your web browser allows the use of cookies.

Good luck and have fun!
